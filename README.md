# slider-design

An image slider/carousel with 3D transitions and text overlays.

## Features

- [x] Auto-advances slides every X seconds
- [x] Previous and next navigation buttons
- [x] **Bonus:** Smooth 3D slide transition animation
- [x] **Bonus:** Text overlay with tag, title, and description on each slide
- [x] **Bonus:** 3D rotateY effect on slide transition
- [x] Dot indicators for direct navigation
- [x] Slide counter (e.g. 1 / 5)
- [x] Progress bar showing time until next slide
- [x] Adjustable interval (2–10 seconds)
- [x] Pause / Resume button
- [x] Keyboard arrow key navigation
- [x] Touch swipe support

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)

## How to run

Open `index.html` in your browser — no build tools needed.
